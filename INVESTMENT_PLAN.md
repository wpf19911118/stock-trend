# 中长期投资指标系统规划（6个月~1年调仓策略）

## 一、问题诊断

**当前双均线策略的局限性：**
- ✅ 优点：简单易懂，趋势跟踪效果好
- ❌ 缺点：滞后性强，震荡市频繁假信号，缺乏估值和宏观判断
- 📊 适合：短期趋势判断（1-3个月）
- 🎯 不适合：中长期资产配置决策

## 四、多因子评分系统

### 4.1 评分规则（总分100分）

```
综合得分 = 技术面(30分) + 估值面(35分) + 宏观面(25分) + 情绪面(10分)
```

### 4.2 评分细则

#### 技术面评分（满分30分）

| 指标 | 评分标准 | 权重 |
|-----|---------|------|
| 均线趋势 | 50日>200日且向上=10分；50日>200日=5分；死叉=0分 | 10分 |
| MACD | 金叉且>0轴=10分；金叉=5分；死叉=0分 | 10分 |
| RSI | 30-50区间=5分；<30=10分；>70=0分 | 5分 |
| 布林带 | 下轨附近=5分；中轨=3分；上轨=0分 | 5分 |

#### 估值面评分（满分35分）⚠️ 核心

| 指标 | 评分标准 | 权重 |
|-----|---------|------|
| PE分位 | <30%=15分；30-50%=10分；50-70%=5分；>70%=0分 | 15分 |
| PB分位 | <30%=10分；30-50%=7分；50-70%=3分；>70%=0分 | 10分 |
| 股债性价比 | >4%=10分；3-4%=7分；2-3%=3分；<2%=0分 | 10分 |

#### 宏观面评分（满分25分）

| 指标 | 评分标准 | 权重 |
|-----|---------|------|
| 利率环境 | 降息周期=10分；维持=5分；加息=0分 | 10分 |
| 美元指数 | 走弱=8分；震荡=4分；走强=0分 | 8分 |
| VIX | >30=7分；20-30=4分；<20=0分 | 7分 |

#### 情绪面评分（满分10分）

| 指标 | 评分标准 | 权重 |
|-----|---------|------|
| 融资余额 | 低位回升=4分；维持=2分；高位=0分 | 4分 |
| 北向资金 | 7日净流入=3分；震荡=1分；流出=0分 | 3分 |
| 成交量 | 缩量后放量=3分；正常=1分；缩量=0分 | 3分 |

### 4.3 综合评级标准

| 总分区间 | 评级 | 操作建议 | 仓位建议 |
|---------|------|---------|---------|
| 80-100分 | **A级（强烈买入）** | 超配，积极加仓 | 90-100% |
| 65-79分 | **B级（买入）** | 标配，可建仓 | 70-90% |
| 50-64分 | **C级（持有）** | 中性，维持现状 | 50-70% |
| 35-49分 | **D级（观望）** | 减仓，等待机会 | 30-50% |
| 0-34分 | **E级（卖出）** | 清仓或做空 | 0-30% |

## 五、调仓触发条件

### 5.1 调仓周期

```python
# 主要调仓时点：每季度（3/6/9/12月末）
# 紧急调仓：当评分跨级（如B→D或D→B）时触发
# 日常监控：每日评分，每周推送简报
```

### 5.2 买卖信号规则

#### 买入信号（满足任意3项以上）

1. ✅ 综合评分 ≥ B级（65分）且趋势向上
2. ✅ PE/PB处于历史30%分位以下
3. ✅ 股债性价比 > 3.5%
4. ✅ 50日/200日均线金叉且MACD确认
5. ✅ VIX > 30（恐慌时逆向布局）
6. ✅ 北向资金连续7日净流入

#### 卖出信号（满足任意2项以上）

1. ⚠️ 综合评分下降至 D级（<50分）
2. ⚠️ PE/PB处于历史70%分位以上
3. ⚠️ 股债性价比 < 2%
4. ⚠️ 50日/200日均线死叉
5. ⚠️ RSI > 80 且放量滞涨
6. ⚠️ 美联储进入加息周期

### 5.3 调仓执行策略

```python
# 渐进式调仓（避免一次性重仓）
评分 A级：分3次建仓，每次间隔5个交易日，每次33%
评分 B级：分2次建仓，每次间隔5个交易日，每次50%
评分 D级：分2次减仓，每次50%
评分 E级：一次性清仓
```

## 九、风险提示

⚠️ **本系统为辅助决策工具，不构成投资建议**

1. **模型风险**：历史数据不代表未来表现
2. **数据延迟**：免费数据源可能有延迟
3. **黑天鹅事件**：极端市场条件下模型可能失效
4. **汇率风险**：美股投资需考虑汇率波动
5. **政策风险**：A股和美股受不同政策影响
