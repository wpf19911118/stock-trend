| PE/PB分位 | 估值水平 | 评分 |
|----------|---------|------|

### ✅ 2. 多因子评分系统 V2 (`scoring_system_v2.py`)

**优化内容：**

#### 技术面评分（30分 → 优化为更精细）
- **均线趋势** (12分): 价格>50日>200日 = 12分（最高权重）
- **MACD** (8分): 金叉+零轴上方 = 8分
- **RSI** (5分): 超卖区间 = 5分
- **布林带** (5分): 下轨附近 = 5分

#### 估值面评分（35分）
- **PE分位** (15分): 基于历史分位
- **PB分位** (10分): 基于历史分位
- **股债性价比** (10分): 风险溢价 >4% = 10分

#### 宏观面评分（25分）
- **利率环境** (10分): 区分A股/美股不同判断
- **VIX** (8分): >35极度恐慌 = 8分
- **美元指数** (7分): 影响新兴市场资金流向

#### 情绪面评分（10分）
- **短期动量** (4分): 5日收益
- **波动率** (3分): 年化波动率
- **趋势强度** (3分): 均线斜率

### ✅ 3. 数据缓存模块 (`data_cache.py`)

**功能特性：**
- JSON数据缓存（API响应）

- 自动过期清理（默认6小时）
- 手动清理接口

**缓存策略：**


**优势：**
- 减少API调用次数
- 加快重复运行速度
- 降低被封禁风险

## 🔧 技术改进

### 评分算法对比

| 维度 | V1.0 | V2.0 |
|-----|------|------|
| **技术指标** | 仅双均线 | MA/MACD/RSI/布林带 |
| **估值指标** | ❌ 无 | ✅ PE/PB分位+股债性价比 |
| **宏观指标** | ❌ 无 | ✅ 利率/VIX/美元指数 |
| **情绪指标** | ❌ 无 | ✅ 动量/波动率/趋势强度 |
| **数据缓存** | ❌ 无 | ✅ 6小时缓存 |
| **估值分析** | ❌ 无 | ✅ 历史分位计算 |

### 评分精度提升

- **估值权重增加**: 从简单判断 → 历史分位精细评分
- **宏观因素加入**: 考虑利率环境和市场恐慌度
- **多维度验证**: 技术面+估值面双重确认，减少假信号
