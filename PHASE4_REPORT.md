### 2. 投资组合管理 (`portfolio_manager.py`)

**功能特性：**
- 💼 持仓跟踪（份额、成本、当前价）
- 💰 自动盈亏计算（已实现/未实现）
- 📝 完整交易记录（买入/卖出/原因）
- 🔄 一键调仓到目标配置
- 💾 数据持久化（JSON存储）

**核心功能：**
```python
# 初始化
portfolio.initialize(1000000)

# 买入
portfolio.buy(symbol, name, market, shares, price, reason)

# 卖出
portfolio.sell(symbol, shares, price, reason)

# 调仓
portfolio.rebalance(target_allocations, current_prices)
```

**输出报告：**
- 总资产价值
- 已实现/未实现盈亏
- 持仓详情（市值、权重、盈亏）
- 交易历史

### 3. 智能告警系统 (`alert_system.py`)

**告警类型：**
- 🔄 **调仓提醒** - 定期调仓、跨级变化
- 📊 **评分极值** - 极高（>85分）/ 极低（<25分）
- 📉 **风险告警** - 快速下跌、VIX恐慌
- ℹ️ **系统告警** - 数据异常、运行错误

**告警级别：**
- 🚨 **Critical** - 严重（立即处理）
- ⚠️ **High** - 高（24小时内处理）
- 📢 **Medium** - 中（关注）
- ℹ️ **Low** - 低（参考）
- 💬 **Info** - 信息

**智能特性：**
- ⏱️ **冷却机制** - 避免重复告警（12-168小时可配置）
- 🎯 **条件触发** - 灵活的Lambda函数条件
- 📱 **微信推送** - 自动发送告警通知
- 📚 **历史记录** - 完整的告警历史

**默认规则：**
1. 评级跨级变化（≥2级）- High
2. 极高评分（≥85分）- Medium，72小时冷却
3. 极低评分（≤25分）- High，72小时冷却
4. 快速下跌（7日-10分）- High
5. 调仓周期（≥90天）- Medium，周冷却
6. VIX恐慌（≥35）- High

---

### 4. 风险管理 (`risk_manager.py`)

**风险指标：**
- 📊 **波动率** - 20日/60日年化波动率
- 📉 **最大回撤** - 90日最大回撤
- 🎲 **VaR** - 95%/99%置信度风险价值
- 📈 **Beta** - 系统性风险（相对于市场）
- 🎯 **风险评级** - 低/中/高/极高

**风控功能：**
- 🛑 **止损** - 固定10%止损线
- 🎯 **止盈** - 固定20%止盈线
- 🔄 **移动止损** - 8%回撤触发
- ⚖️ **仓位控制** - 风险自适应仓位建议

**仓位建议逻辑：**
```python
风险极高 → 减仓至5%
风险高 + 仓位>30% → 减仓至30%
评分高(≥80) + 风险低 → 加仓
评分低(<35) → 减仓
```
